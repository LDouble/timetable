# 常用命令
- `ls`, `rg <pattern>`：查看目录与快速搜索源码（推荐 ripgrep）
- `git status` / `git diff`：查看工作区与提交差异
- `flutter pub get`：安装/同步 Flutter 依赖
- `flutter analyze`：运行静态检查（依赖 `flutter_lints` 规则）
- `flutter test`：执行单元/组件测试
- `dart format .`（或 `flutter format`）：统一代码风格
- （参考项目）在 `ColorTimetable-main/` 下使用 `pnpm install` 与 `pnpm run dev:mp-weixin` 来运行原 Vue 版本以对照 UI 效果
